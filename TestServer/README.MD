# TestServer
这是一个基于Flask框架的Server,API遵循Restful规范。

##### 1.项目依赖

```
pip==20.0.2
PyMySQL==0.9.3
Flask==1.1.1
Flask-SQLAlchemy==2.4.1
SQLAlchemy==1.3.13
SQLAlchemy-Utils==0.36.1
```

 安装依赖，请切换到项目工程目录下面。然后使用如下命令进行安装：

```
pip install -r requirements.txt
```

##### 2. 项目的启动文件

```
manage.py
```
